用到的npm库介绍 第一期 (每周15库)

计划整理一下会用到的一些npm库，每周做一期。
希望能够使用这个办法扩展思路，了解到更多的js技术方向。
如果关于这个注意你有更好的想法，欢迎在这里讨论

  1. 关键字提示字符截取：
    isaacs/abbrev-js: Like ruby’s Abbrev module

  2. terminal 颜色：
    doowb/ansi-colors: Easily add ANSI colors to your text and symbols in the terminal. Used by webpack, gulp, and many others!

  3. accepts 这条线路没有搞明白。内容协商？
    accepts - npm
    negotiator - npm

  4. 编译语言模板统一结构，没有理解 :smiley:
    accord - npm

  5. 一个小型的js的解析器，有解析jsx 的插件，看起来非常不错，要是自己写模板解析器估计可以参考这个东西 :+1:
    acorn - npm

  6. zip 数据压缩方向
    adm-zip - npm

  7. after 没有考虑明白的一个库, 流程控制？回调函数？
    after - npm

  8. agent-base 一个过于http.agent 的一个扩展
    agent-base - npm

  9. ajv json 结构验证器，可以用于浏览器
    ajv - npm

  10. align-text 对齐字符串
    align-text - npm

  11. alphanum-sort 字母排序算法
    alphanum-sort - npm

  12. 用来实现atob btoa 64位解码
    abab - npm

  13. Get current machine IP, MAC and DNS servers.
    address - npm

  14. 做文本匹配，看起来比较常用在路径匹配方向上
    anymatch - npm

  15. append-transform，我并没有很好的理解这个意思
    append-transform - npm
